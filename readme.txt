YouTube Gallery plugin to display YouTube Channel or Playlist videos.

== Description ==

Best Responsive YouTube Gallery plugin for WordPress to display YouTube Channel or Playlist videos at your site. Display anywhere at your site using shortcode [youtube_gallery"]. YouTube Gallery plugin packed with necessary controlling options.

YouTube Gallery plugin is simple, flexible & powerful.

### Plugin Features
* Responsive YouTube Gallery plugin
* Pull latest videos from any YouTube Channel
* Choose the number of videos to display
* Different Column options
* Display YouTube videos anywhere with the shortcode [youtube_gallery].
* Options for 2 columns, 3 columns & 4 columns
* Custom CSS
* Works with any standard WordPress Theme.
* Developer friendly & easy to customize.
* All modern browsers supported.
* W3C valid markup
* Easy to setup

== Installation ==

Install YouTube Gallery plugin as a regular WordPress plugin. Here is different ways to install YouTube Gallery plugin :

= Install by Search Plugin =
- In your Admin, go to menu Plugins > Add New
- Search - YouTube Gallery
- Click to install
- Activate the plugin

= Install by Upload Plugin =
- Download the latest version of YouTube Gallery (.zip file)
- In your Admin, go to menu Plugins > Add New
- Select "Upload Plugin"
- Click on "Choose File"
- Select downloaded youtube-gallery.zip & click on 'Install Now' button
- Activate the plugin

= Install Plugin using FTP =
- Upload youtube-gallery.zip folder inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins & Activate the YouTube Gallery plugin

== Frequently Asked Questions ==
= How to use shortcode inside page templates? =
WordPress has a great function, do_shortcode(), that will allow you to use shortcodes inside your theme files. For example, to output Youtube Videos in a Theme file, you would do this: 
`
<?php echo do_shortcode( '[youtube_gallery]' ); ?>
`
